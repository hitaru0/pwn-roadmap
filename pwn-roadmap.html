Kernel Protections & Bypasses
SMEP, SMAP, KASLR, KPTI — understand what each stops and how to work around it.
SMEPSMAPKASLRKPTI
  • SMEP: blocks kernel executing user-space pages
  • SMAP: blocks kernel reading/writing user-space pages
  • KASLR: randomizes kernel base (leak via /proc or timing)
  • KPTI: separate page tables for user/kernel
  • Kernel ROP: disable SMEP via CR4, then ret2usr
  • KROP with gadgets from kernel .text
